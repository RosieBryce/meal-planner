# How to run Rosie Chef

1. Open Terminal (search "Terminal" or "PowerShell" in the Start menu)
2. Navigate to the project folder:
   
   ```
   cd "C:\Users\DELL\My Drive\Workspace\Code\green-chef"
   ```
3. Start the app:
   
   ```
   npm run dev
   ```
4. Open your browser and go to: **http://localhost:5173**

To stop it, press `Ctrl + C` in the terminal.

---

**Only needed once** — if you've never run it before or get an error about missing modules:

```
npm install
```

Then run `npm run dev` again.
